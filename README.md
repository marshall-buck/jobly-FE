# jobly

### React front end  that luse a backend job api



#### Tech Stack


- React Front end
- Typescript
- React testing lib
- Cypress
- Tailwindcss
- DaisyUi
---

#### Features

- users can register, login/out
- protected auth routes
- users can search for jobs
- properties won't double book
